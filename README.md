# JS Editor

A browser-based TypeScript/JavaScript code editor with live output preview.

## Features

- Split-pane layout with resizable editor and output
- Monaco-style code editing with CodeMirror
- TypeScript support with real-time transpilation
- JavaScript support
- Console output with timestamps (log, error, warn, info)
- Dark/light theme toggle
- Auto-save to localStorage
- Keyboard shortcut: Ctrl+Enter to run

## Usage

Open `index.html` in a browser to use the editor.

## Deploy

This project is deployed on GitHub Pages.
